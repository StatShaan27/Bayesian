About This Project

This project is carried out under the guidance of Prof. Sourabh Bhattacharya. The main objective is to critically compare the Bayesian, Classical, and Machine Learning paradigms, highlighting paradoxes in the latter two and showing how Bayesian methods resolve them, both in theory and in practice.
